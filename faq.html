General Questions

What is Tome? Tome is a roleplaying support addon for RIFT that allows players to add and display additional information about their characters such as appearance, prefixes/suffixes, and background.
Is Tome compatible with Merisioux? Yes.
Will Tome interfere with any other addons? No.
Can I run Tome and Merisioux at the same time? Yes, but users who are only running Merisioux may experience display issues when viewing your data, especially if you are transmitting data from both addons.

Tome does pretty much everything that Merisioux does, so there's not really much point in running both addons together, anyway.
Can I test Tome? Yes. You can download it here.

Features

What's Tutor Mode? Tutor Mode is a quick and easy way for RPers to broadcast themselves as willing to help out beginner roleplayers. If an inexperienced roleplayer sees the tutor flag, they know they can come to that person with any questions or ask for advice.

As such, we would suggest that you only toggle on Tutor Mode if you're willing to be helpful and beneficial to new roleplayers in showing them the ropes and answering their questions.
What are the RP flags? The RP flags, or the experience flags, are the items contained within the dropdown menu on the character edit pane. These are used to display your experience and familiarity with RP.

  • Non-Roleplayer
    Pretty self explanatory here.
  • Fledgling Roleplayer
    This is for newer roleplayers, either to RP as a whole, or possibly just RIFT. These are the guys Tutor Mode is designed to help.
  • Roleplayer
    You've been around roleplay enough to consider yourself a roleplayer, but you don't consider yourself to be experienced yet.
  • Experienced Roleplayer
    You've been roleplaying enough to know the ins and outs, the etiquette and the places to be.

Common Issues

Why are the Guild and Settings tab empty? Both the Guild tab and Settings tab have not yet been given any features. The buttons are merely there as placeholders until we finish up the content for those tabs.
The dropdown menu is overlapped by the text area! I can't select the option I want. The current beta build of Tome (0.2-beta) has not been updated with this fix yet. A workaround is available, however. Firstly try typing "/reloadui" into the chatbox in game.

If that doesn't solve your issue, you can force Tome to display the option you want via command line. Type "/tome set flag x" into the chatbox in game, where the x is the experience level you want, as follows:

0 = Non-Roleplayer
1 = Flegling Roleplayer
2 = Roleplayer
3 = Experienced Roleplayer
Why isn't there text wrapping in the Edit Pane? You have it in the View Pane. It's physically, metaphorically, literally, figuratively and theoretically impossible for us to add text wrapping into the Edit Pane. This is a limitation on the addon resources we are given by Trion.
Manually entering line breaks and indents work just fine and will be displayed correctly in the Edit Pane, however.
I updated my Tome and clicked save. Why can't my friend see my changes? In the current version of Tome (0.2-beta) updates are not broadcast when the save button is pressed, they are only saved locally. In order for your updates to be seen, the viewer must clear their Tome cache by either waiting an hour for the cache to refresh or by manually clearing the cache by typing "/tome clear" into the chatbox in game.
I can't type in the Appearance or History fields! When trying to select the Appearance or History fields, make sure that you click on the top line. Clicking anywhere else while the field is blank will not enable the field.
Tabbed indents give me an error / move the cursor an extra space forward. This is an error in an older version of Tome. To fix it, please download the latest version.
If you have the latest version (uploaded 15th July), please create an issue in the issue tracker, or contact Maiiel@Faeblight in game.
Blank new lines are deleted when text is added to a new line below. This is an error in an older version of Tome. To fix it, please download the latest version.
If you have the latest version (uploaded 15th July), please create an issue in the issue tracker, or contact Maiiel@Faeblight in game.
x feature isn't working! What can I do? Look for a workaround in the command line by typing "/tome help" into the chatbox in game. If you can't find a workaround, contact Maiiel@Faeblight with the issue and they'll either help you get it sorted, or make a note of the bug and add it to the list of stuff that needs to be fixed.

We're not perfect.
